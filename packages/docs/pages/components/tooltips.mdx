## Import

```js
import { Tooltip } from '@cube/components';
```

## Usage

Pass the component that you want to give the tooltip to as a child of the `Tooltip` component. `Tooltip` content can be set using the `content` prop.

<Example>
  <Tooltip content="This is a tooltip">
    <Button>Hover for a tooltip</Button>
  </Tooltip>
</Example>

```jsx /content/
<Tooltip content="This is a tooltip">
  <Button>Hover for a tooltip</Button>
</Tooltip>
```

## Event

Use `showOn` prop to customize the tooltip to show on `hover` or `click`.

<Example>
  <Tooltip showOn="hover" content="This is a tooltip">
    <Button>Hover for a tooltip</Button>
  </Tooltip>
  <DeleteConfirmation />
</Example>

```jsx /showOn/
<Tooltip showOn="hover" content="This is a tooltip">
  <Button>Hover for a tooltip</Button>
</Tooltip>

<Tooltip
  showOn="click"
  content={(close) => (
    <div className="block space-y-5">
      <p className="text-sm text-gray-700">
        Are you sure you want to delete the blog post?
      </p>
      <div className="flex space-x-3">
        <Button
          color="error"
          area="sm"
          variant="light"
          onClick={close}
          className="w-full"
        >
          Confirm deletion
        </Button>
        <Button
          area="sm"
          color="error"
          variant="light"
          onClick={close}
          className="hover:bg-gray-100 bg-gray-50 w-full text-gray-700"
        >
          cancel
        </Button>
      </div>
    </div>
  )}
  className="p-4 text-gray-900 bg-white border border-gray-200 rounded-lg"
>
  <Button color="error">Click to delete Blog post</Button>
</Tooltip>
```

## Transition

Use `transition` property to customize the transition property of the `Tooltip`. Ensure the `duration` of the `transition` property matches the css transition duration.

<Example>
  <Tooltip
    content="This is a tooltip with customized transition"
    className="duration-200 transform"
    transition={{
      duration: 200,
      enter: 'translate-x-0 opacity-100',
      exit: 'translate-x-10 opacity-0',
    }}
  >
    <Button>Hover for a tooltip</Button>
  </Tooltip>
</Example>

```jsx
<Tooltip
  content="This is a tooltip with customized transition"
  className="duration-200 transform"
  transition={{
    duration: 200,
    enter: 'translate-x-0 opacity-100',
    exit: 'translate-x-10 opacity-0',
  }}
>
  <Button>Hover for a tooltip</Button>
</Tooltip>
```
