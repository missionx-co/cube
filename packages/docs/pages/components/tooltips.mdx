## Usage

`Tooltip` implementation is totally based on `@tippy/react` component. Visit [their documentation](https://github.com/atomiks/tippyjs-react) for more details on props and customiation

<Example><Tooltip content="This is a tooltip">
  <Button>Hover for a tooltip</Button>
</Tooltip>

  <DeleteConfirmation />
</Example>

```jsx
const [visible, setVisible] = useState(false);
const show = () => setVisible(true);
const hide = () => setVisible(false);

<Tooltip content="This is a tooltip">
  <Button>Hover for a tooltip</Button>
</Tooltip>

<Tooltip
  interactive
  visible={visible}
  content={
    <div className="block space-y-5">
      <p className="text-sm text-gray-700">
        Are you sure you want to delete the blog post?
      </p>
      <div className="flex space-x-3">
        <Button
          color="error"
          area="sm"
          variant="light"
          onClick={hide}
          className="w-full"
        >
          Confirm deletion
        </Button>
        <Button
          area="sm"
          color="error"
          variant="light"
          onClick={hide}
          className="hover:bg-gray-100 bg-gray-50 w-full text-gray-700"
        >
          cancel
        </Button>
      </div>
    </div>
  }
  className="p-4 text-gray-900 bg-white border border-gray-200 rounded-lg"
  onClickOutside={hide}
>
  <Button color="error" onClick={show}>
    Delete Blog post
  </Button>
</Tooltip>
```
