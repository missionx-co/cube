## Import

```js
import { Input } from '@cube-ui/components';
```

## Usage

Textarea is a subcomponent of the `Input` component.

<Example>
   <Input.Textarea placeholder="jhon.doe@example.com" />

  <Input.Textarea placeholder="jhon.doe@example.com" disabled />
</Example>

```jsx
<Input.Textarea placeholder="jhon.doe@example.com" />

<Input.Textarea placeholder="jhon.doe@example.com" disabled />
```

## Component API

The `Input.Textarea` accepts any prop that a normal html `textarea` can have. Plus the `error` prop.
