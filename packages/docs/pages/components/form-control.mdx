## Import

```jsx
import { FormControl } from '@cube/components';
```

## Usage

<Example>
  <FormControl
    id="email-1"
    fieldLabel="Email"
    hint="Enter your email."
  >
    <Input
      id="email-1"
      name="email-1"
      placeholder="jhon.doe@example.com"
    />
  </FormControl>

  <FormControl
    id="email-2"
    fieldLabel="Email"
    hint="Enter your email."
    error="Email is invalid"
  >
    <Input
      id="email-2"
      name="email-2"
      placeholder="jhon.doe@example.com"
      error
    />
  </FormControl>
</Example>

## Props

`FormControl` accepts any props that a normal html `div` can have. Plus, the following props

<PropsList
  props={[
    {
      name: 'id',
      type: 'string',
      description:
        '(Required). The id of the input that will be used as a form control child. this id will be passed to the label `htmlFor` property',
    },
    {
      name: 'fieldLabel',
      type: 'ReactNode',
      description: 'Form control label',
    },
    {
      name: 'hint',
      type: 'ReactNode',
      description: 'Form control hint text.',
    },
    {
      name: 'error',
      type: 'ReactNode',
      description:
        'Form control error message. When error property is set. The error message will appear instead of the hint text.',
    },
    {
      name: 'atoms',
      type: `AtomsProps`,
      description:
        'Properties to be passed to the Form control label, hint text or error message.',
    },
  ]}
/>

```ts
interface AtomsProps {
  label?: Omit<HTMLProps<HTMLLabelElement>, 'children'>;
  hint?: Omit<HTMLProps<HTMLSpanElement>, 'children'>;
  error?: Omit<HTMLProps<HTMLSpanElement>, 'children'>;
}
```
