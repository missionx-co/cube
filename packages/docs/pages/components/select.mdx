## Import

```js
import { Select } from '@cube-ui/components';
```

## Usage

<BasicUsage />

```jsx
const options = [
  ...
  id: 'three',
    text: 'Option three',
    disabled: true,
  },
  {
    id: 'group',
    text: 'Options group',
    children: [
      {
        id: 'group-one',
        value: 'group-one',
        text: 'Option group - option one',
      },
      {
        id: 'group-two',
        value: 'group-two',
        text: 'Option group - option two',
      },
    ],
  },
  ...
];

<Select placeholder="Please select an option" options={options} />
<Select placeholder="Please select an option" options={options} disabled />
<Select placeholder="Please select an option" options={options} error />
```

## Component API

The `Select` accepts any prop that a normal html `select` can have. Plus the `error` prop.

<PropsList
  props={[
    {
      name: 'error',
      type: 'boolean',
      description: 'Indicates if input field has an invalid state.',
      default: 'false',
    },
    {
      name: 'options',
      type: 'Option[]',
    },
  ]}
/>

```jsx
export interface Option {
  id: Key;
  text: string;
  disabled?: boolean;
  children?: Option[];
}
```
