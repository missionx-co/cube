## Colors

Colors can be added to the `Badge` component with the `color` prop.

Cube supports 8 colors for the `Badge` component: `primary`, `error`, `success`, `warning`, `blue-gray`, `blue-light`, `blue`, `indigo`, `purple`, `pink`, `rose` and `orange`.

## Size

Badge size can be defined with the `area` prop.

Cube supports three sizes for the `Badge` component: `sm`, `base` and `lg`

<Example>
  <Badge area="sm">Label</Badge>
  <Badge>Label</Badge>
  <Badge area="lg">Label</Badge>
</Example>

```js
<Badge area="sm">Label</Badge>
<Badge>Label</Badge>
<Badge area="lg">Label</Badge>
```
