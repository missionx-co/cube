## Import

```js

```

## Usage

<Example>
  <Switch>
    <Switch.Label>Receive email notifications</Switch.Label>
  </Switch>

  <Switch disabled>
    <Switch.Label>Forget my information</Switch.Label>
  </Switch>
</Example>

```jsx

```

## Size

You can change the `Switch` size with the area prop. It accepts one of: `base` or `sm`.

<Example>
  <Switch area="base">
    <Switch.Label>Remember me</Switch.Label>
  </Switch>

  <Switch area="sm">
    <Switch.Label>Remember me (small)</Switch.Label>
  </Switch>
</Example>

```jsx
<Switch area="base">
  <Switch.Label>Remember me</Switch.Label>
</Switch>

<Switch area="sm">
  <Switch.Label>Remember me (small)</Switch.Label>
</Switch>
```
