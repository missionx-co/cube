## Import

```js
import { FancySelect2 } from '@cube/components';
```

## Customization

use `className` property to pass css classes to customize the `Input` component however you want.

<Example>
  <div className="flex w-96">
    <span className="px-3.5 py-2.5 bg-gray-50 border border-gray-300 rounded-l-lg">
      https://
    </span>
    <Input
      placeholder="mohammedmanssour.me"
      className="border-l-0 rounded-l-none"
    />
  </div>
</Example>

```jsx
<div className="flex w-96">
  <span className="px-3.5 py-2.5 bg-gray-50 border border-gray-300 rounded-l-lg">
    https://
  </span>
  <Input
    placeholder="mohammedmanssour.me"
    className="border-l-0 rounded-l-none"
  />
</div>
```
