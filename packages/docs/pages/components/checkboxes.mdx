## Import

```js
import { Checkbox } from '@cube/components';
```

## Usage

<Example>
  <Checkbox
    onChange={(checked) => {
      console.log(checked);
    }}
  >
    <Checkbox.Label>Remember me</Checkbox.Label>
  </Checkbox>

  <Checkbox disabled>
    <Checkbox.Label>Remember me (disabled)</Checkbox.Label>
  </Checkbox>
</Example>

```jsx
<Checkbox
  onChange={(checked) => {
    console.log(checked);
  }}
>
  <Checkbox.Label>Remember me</Checkbox.Label>
</Checkbox>
```

## Size

You can change the `Checkbox` size with the area prop. It accepts one of: `base` or `sm`.

<Example>
  <Checkbox>
    <Checkbox.Label>Remember me</Checkbox.Label>
  </Checkbox>

  <Checkbox area="sm">
    <Checkbox.Label>Remember me (small)</Checkbox.Label>
  </Checkbox>
</Example>

```jsx
<Checkbox area="base">
  <Checkbox.Label>Remember me</Checkbox.Label>
</Checkbox>

<Checkbox area="sm">
  <Checkbox.Label>Remember me (small)</Checkbox.Label>
</Checkbox>
```

## Variants

The Checkbox component has two variants: `filled` and `outline`

<Example>
  <Checkbox variant="outline" defaultChecked>
    <Checkbox.Label>
      Remember me (outline variant)
    </Checkbox.Label>
  </Checkbox>

  <Checkbox variant="filled" defaultChecked>
    <Checkbox.Label>
      Remember me (filled variant)
    </Checkbox.Label>
  </Checkbox>
</Example>

```jsx
<Checkbox variant="outline" defaultChecked>
  <Checkbox.Label>
    Remember me (outline variant)
  </Checkbox.Label>
</Checkbox>

<Checkbox variant="filled" defaultChecked>
  <Checkbox.Label>
    Remember me (filled variant)
  </Checkbox.Label>
</Checkbox>
```

## Customization

Use `containerClassName` to customize the container style.

Use `className` to customize the `Checkbox` component style.

Use `icon` to customize the selected icon.

<Example>
  <Checkbox className="items-center w-10 h-10 p-1 rounded-full">
    <Checkbox.Label className="flex flex-col ml-4">
      <span>Remember me</span>
      <span className="font-normal text-gray-500">
        Save my login details for next time.
      </span>
    </Checkbox.Label>
  </Checkbox>
</Example>

```jsx
<Checkbox className="items-center w-10 h-10 p-1 rounded-full">
  <Checkbox.Label className="flex flex-col ml-4">
    <span>Remember me</span>
    <span className="font-normal text-gray-500">
      Save my login details for next time.
    </span>
  </Checkbox.Label>
</Checkbox>
```
