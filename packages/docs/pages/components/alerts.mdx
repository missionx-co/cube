
<Example>

## Colors

Colors can be added to the `Alert` component with the `color` prop.

Cube supports four colors for the `Alert` component: `primary`, `error`, `success` and `warning`

<Example>
  <div className="flex flex-col space-y-3 w-full">
    <Alert>Change a few things up and try submitting again.</Alert>
    <Alert color="success">
      <span>Change a few things up and try submitting again.</span>
    </Alert>
    <Alert color="error">
      <span>Change a few things up and try submitting again.</span>
    </Alert>
    <Alert color="warning">
      <span>Change a few things up and try submitting again.</span>
    </Alert>
  </div>
</Example>

```js
<Alert>Change a few things up and try submitting again.</Alert>
<Alert color="success">Change a few things up and try submitting again.</Alert>
<Alert color="error">Change a few things up and try submitting again.</Alert>
<Alert color="warning">Change a few things up and try submitting again.</Alert>
```

## Variants

Variants can be added to the `Alert` with the `variant` prop.

Cube supports two variants for the `Alert` component: `filled` and `light`. The example above has the light version

<Example>
  <div className="flex flex-col space-y-3 w-full">
    <Alert variant="filled">
      <span>Change a few things up and try submitting again.</span>
    </Alert>
    <Alert variant="filled" color="success">
      <span>Change a few things up and try submitting again.</span>
    </Alert>
    <Alert variant="filled" color="error">
      <span>Change a few things up and try submitting again.</span>
    </Alert>
    <Alert variant="filled" color="warning">
      <span>Change a few things up and try submitting again.</span>
    </Alert>
  </div>
</Example>

```jsx
<Alert variant="filled">
  Change a few things up and try submitting again.
</Alert>
<Alert variant="filled" color="success">
  Change a few things up and try submitting again.
</Alert>
<Alert variant="filled" color="error">
  Change a few things up and try submitting again.
</Alert>
<Alert variant="filled" color="warning">
  Change a few things up and try submitting again.
</Alert>
```

## Alert.Title

Alert provide a subcomponent `Alert.Title` that has a specific styling that you can use as an Alert title.

<Example>
  <Alert>
    <Alert.Title>Info alert!</Alert.Title>
    <span>Change a few things up and try submitting again.</span>
  </Alert>
</Example>

```jsx
<Alert>
  <Alert.Title>Info alert!</Alert.Title>
  <span>Change a few things up and try submitting again.</span>
</Alert>
```
