## Import

```js
import { Radio } from '@cube-ui/components';
```

## Usage

Group of `Radio`s must be wraped with a `Radio.Group`.

<Example>
  <Radio.Group name="size" orientation="horizontal">
    <Radio value="remeber-me">
      <Radio.Label>Remember me</Radio.Label>
    </Radio>

    <Radio value="delete-data">
      <Radio.Label>Delete my data after logout</Radio.Label>
    </Radio>

    <Radio value="ignore" disabled>
      <Radio.Label>Do nothing</Radio.Label>
    </Radio>

  </Radio.Group>
</Example>

```jsx
<Radio.Group name="size" orientation="horizontal">
  <Radio value="remeber-me">
    <Radio.Label>Remember me</Radio.Label>
  </Radio>

  <Radio value="delete-data">
    <Radio.Label>Delete my data after logout</Radio.Label>
  </Radio>

  <Radio value="ignore" disabled>
    <Radio.Label>Do nothing</Radio.Label>
  </Radio>
</Radio.Group>
```

## Size

You can change the `Radio` size with the area prop. It accepts one of: `base` or `sm`.

<Example>
  <Radio.Group name="size" orientation="horizontal">
    <Radio value="remeber-me">
      <Radio.Label>Remember me</Radio.Label>
    </Radio>
    <Radio value="remember-me-sm" area="sm">
      <Radio.Label>Remember me (small)</Radio.Label>
    </Radio>
  </Radio.Group>
</Example>

```jsx
<Radio.Group name="size" orientation="horizontal">
  <Radio value="remeber-me" area="base">
    <Radio.Label>Remember me</Radio.Label>
  </Radio>

  <Radio value="remember-me-sm" area="sm">
    <Radio.Label>Remember me (small)</Radio.Label>
  </Radio>
</Radio.Group>
```

## Variants

The `Radio` component has two variants: `filled` and `outline`

<Example>
  <Radio.Group name="variant" orientation="horizontal">
    <Radio value="outline" variant="outline">
      <Radio.Label>Remember me (outline variant)</Radio.Label>
    </Radio>

    <Radio value="filled" variant="filled">
      <Radio.Label>Remember me (filled variant)</Radio.Label>
    </Radio>

  </Radio.Group>
</Example>

```jsx
<Radio.Group name="variant" orientation="horizontal">
  <Radio value="outline" variant="outline">
    <Radio.Label>Remember me (outline variant)</Radio.Label>
  </Radio>

  <Radio value="filled" variant="filled">
    <Radio.Label>Remember me (filled variant)</Radio.Label>
  </Radio>
</Radio.Group>
```

## Customization

Use `containerClassName` to customize the container style.

Use `className` to customize the `Radio` component style.

Use `icon` to customize the selected icon.

<Example>
  <Radio.Group name="custom" className="space-y-10">
    {['custom', 'custom-2'].map((name) => (
      <Radio
        key={name}
        value={name}
        className="items-center w-10 h-10 p-1"
        icon={<span className="bg-primary-600 block w-4 h-4 rounded-full" />}
      >
        <Radio.Label className="flex flex-col ml-4">
          <span>Remember me</span>
          <span className="font-normal text-gray-500">
            Save my login details for next time.
          </span>
        </Radio.Label>
      </Radio>
    ))}
  </Radio.Group>
</Example>

```jsx
<Radio.Group name="custom" className="space-y-10">
  {['custom', 'custom-2'].map((name) => (
    <Radio
      key={name}
      value={name}
      className="items-center w-10 h-10 p-1"
      icon={() => (
        <span className="bg-primary-600 block w-4 h-4 rounded-full" />
      )}
    >
      <Radio.Label className="flex flex-col ml-4">
        <span>Remember me</span>
        <span className="font-normal text-gray-500">
          Save my login details for next time.
        </span>
      </Radio.Label>
    </Radio>
  ))}
</Radio.Group>
```
